# Socket-сервер с протоколом для обмена .html файлов между сервером и клиентом

Для запуска серверной части используйте команду ```python server.py```

Для запуска клиентской части используйте команду ```python client.py```

После запуска команд в корневой директории будут созданы папки **html_storage и client_data**
